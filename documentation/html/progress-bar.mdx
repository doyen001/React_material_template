# Tailwind CSS Progress

Our Tailwind CSS progress component can be used to show a user how far along he is in a process. The progress can be determinate or indeterminate. Use the Progress Bar to show an ongoing process that takes a noticeable time to finish.

Below we are presenting our examples of progress components that you can use in your Tailwind CSS project. They come in different styles so you can adapt them easily to your needs.

<br />

<ComponentDemo id="progress">
  <div className="w-full">
    <span className="text-blue-gray-500 text-sm font-normal leading-6">
      {`50%`}
    </span>
    <div className="progress">
      <div className="progress-bar progress-pink" data-value="50"></div>
    </div>
  </div>
</ComponentDemo>

```html
<div class="w-full">
  <span class="text-blue-gray-500 text-sm font-normal leading-6">50%</span>
  <div class="progress">
    <div class="progress-bar progress-pink" data-value="50"></div>
  </div>
</div>
```

## Required Scrips

<span id="required-script" className="scroll-mt-56" />

The progress component needs a required script files to work, you just need to add the below script file to the bottom of your html file.

```html
<!-- from node_modules -->
<script src="node_modules/@material-tailwind/html/scripts/progress.js"></script>

<!-- from cdn -->
<script src="https://unpkg.com/@material-tailwind/html@latest/scripts/progress.js"></script>
```
