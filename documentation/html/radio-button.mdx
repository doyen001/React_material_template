# Tailwind CSS Radio

Our Tailwind CSS radio button component will let your users choose only one of a predefined set of mutually exclusive options. Radio buttons should be used instead of checkboxes if only one item can be selected from a list.

Use the following example to create simple radio buttons for your projects.

<br />

<ComponentDemo id="radio">
  <div className="text-left">
    <div className="form-check mb-4 flex">
      <input
        name="inputName"
        className="radio absolute z-10 cursor-pointer opacity-0"
        type="radio"
        id="flexRadioDefault1"
      />
      <div className="form-check-input"></div>
      <label
        className="custom-control-label my-auto cursor-pointer"
        htmlFor="flexRadioDefault1"
      >
        {`Default radio`}
      </label>
    </div>
    <div className="form-check flex">
      <input
        name="inputName"
        className="radio absolute z-10 cursor-pointer opacity-0"
        type="radio"
        id="flexRadioDefault2"
        defaultChecked
      />
      <div className="form-check-input"></div>
      <label
        className="custom-control-label my-auto cursor-pointer"
        htmlFor="flexRadioDefault2"
      >
        {`Default checked radio`}
      </label>
    </div>
  </div>
</ComponentDemo>
