# Tailwind CSS Avatar - React

Use our Tailwind CSS <Code>Avatar</Code> component to portray people or objects in your web projects. The <Code>Avatar</Code> can be used as a visual identifier for a user profile on your website.

See below our beautiful <Code>Avatar</Code> example that you can use in your Tailwind CSS and React project. The example also comes in different styles and colors so you can adapt it easily to your needs.

<br />

<ComponentDemo id="avatar">
  <Avatar src="/img/face-2.jpg" alt="avatar" />
</ComponentDemo>

```jsx
import { Avatar } from "@material-tailwind/react";

export default function Example() {
  return <Avatar src="/img/face-2.jpg" alt="avatar" />;
}
```

---

## Avatar Variants

The <Code>Avatar</Code> component comes with 2 different variants that you can change it using the <Code>variant</Code> prop.

<ComponentDemo id="avatar-variants">
  <div className="flex gap-4">
    <Avatar src="/img/face-2.jpg" alt="avatar" />
    <Avatar src="/img/face-2.jpg" alt="avatar" variant="circular" />
  </div>
</ComponentDemo>

```jsx
import { Avatar } from "@material-tailwind/react";

export default function Variants() {
  return (
    <div className="flex gap-4">
      <Avatar src="/img/face-2.jpg" alt="avatar" />
      <Avatar src="/img/face-2.jpg" alt="avatar" variant="circular" />
    </div>
  );
}
```
