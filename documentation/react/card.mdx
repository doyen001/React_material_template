# Tailwind CSS Card - React

Use our <Code>Card</Code> to provide a flexible and extensible content container based on Tailwind CSS with multiple variants and options.

By its definition, a <Code>Card</Code> is a sheet of material that serves as an entry point to more detailed information. <Code>Card</Code> usually include a photo, text, and a link about a single subject. They should be easy to scan for relevant and actionable information. Elements, like text and images, should be placed on them in a way that clearly indicates hierarchy.

See below our beautiful <Code>Card</Code> example that you can use in your React and Tailwind CSS projects. We've also included some <Code>Card</Code> props that are available.

export default function Example() {
  return (

          src="/img/blog.jpg"
          alt="img-blur-shadow"

      <CardFooter divider className="flex items-center justify-between py-3">

          <i className="fas fa-map-marker-alt fa-sm mt-[3px]" />
          Barcelona, Spain
        </Typography>
      </CardFooter>
    </Card>
  );
}
