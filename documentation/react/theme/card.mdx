# Tailwind CSS Card - Theme

<span id="overview" className="scroll-mt-48" />

Learn how to customize the theme and styles for card components, the theme object for card components has three main objects:

**A.** The <Code>defaultProps</Code> object for setting up the default value for props of card component.<br />
**B.** The <Code>valid</Code> object for customizing the valid values for card component props.<br />
**C.** The <Code>styles</Code> object for customizing the theme and styles of card component.<br />

You can customize the theme and styles of card components by adding Tailwind CSS classes as key paired values for objects.

<br />
<br />

## Card Theme Object Type

<span id="card-theme-object-type" className="scroll-mt-48" />

```ts
interface CardStylesType {
  defaultProps: {
    variant: string;
    color: string;
    shadow: boolean;
    className: string;
  };
  valid: {
    variants: string[];
    colors: string[];
  };
  styles: {
    base: {
      initial: object;
      shadow: object;
    };
    variants: {
      filled: object;
      gradient: object;
    };
  };
}
```

<br />
<br />

### For TypeScript Only

```tsx
import type { CardStylesType } from "@material-tailwind/react";
```

## Card Theme Customization

<span id="card-theme-customization" className="scroll-mt-48" />

```tsx
const theme = {
  card: {
    styles: {
      base: {
        initial: {
          position: "relative",
        },
      },
    },
  },
};
```
