# Tailwind CSS Dialog - Theme

<span id="overview" className="scroll-mt-48" />

Learn how to customize the theme and styles for dialog components, the theme object for dialog components has three main objects:

**A.** The <Code>defaultProps</Code> object for setting up the default value for props of dialog component.<br />
**B.** The <Code>valid</Code> object for customizing the valid values for dialog component props.<br />
**C.** The <Code>styles</Code> object for customizing the theme and styles of dialog component.<br />

You can customize the theme and styles of dialog components by adding Tailwind CSS classes as key paired values for objects.

<br />
<br />

## Dialog Theme Object Type

<span id="dialog-theme-object-type" className="scroll-mt-48" />

```ts
interface DialogStylesType {
  defaultProps: {
    size: string;
    dismiss: {
      enabled: boolean;
      escapeKey: boolean;
      referencePointerDown: boolean;
      outsidePointerDown: boolean;
      ancestorScroll: boolean;
      bubbles: boolean;
    };
    animate: {
      mount: object;
      unmount: object;
    };
    className: string;
  };
  valid: {
    sizes: string[];
  };
  styles: {
    base: {
      backdrop: object;
      container: object;
    };
    sizes: {
      xs: object;
      sm: object;
      md: object;
      lg: object;
      xl: object;
      xxl: object;
    };
  };
}
```

<br />
<br />

### For TypeScript Only

```tsx
import type { DialogStylesType } from "@material-tailwind/react";
```

## Dialog Theme Customization

<span id="dialog-theme-customization" className="scroll-mt-48" />

```tsx
const theme = {
  dialog: {
    styles: {
      sizes: {
        xs: {
          width: "w-1/4",
          minWidth: "min-w-[25%]",
          maxWidth: "max-w-[25%]",
        },
      },
    },
  },
};
```

## Dialog Body Theme Object Type

<span id="dialog-body-theme-object-type" className="scroll-mt-48" />

```ts
interface DialogBodyStylesType {
  defaultProps: {
    className: string;
    divider: boolean;
  };
  styles: {
    base: {
      initial: object;
      divider: object;
    };
  };
}
```

<br />
<br />

### For TypeScript Only

```tsx
import type { DialogBodyStylesType } from "@material-tailwind/react";
```

## Dialog Body Theme Customization

<span id="dialog-body-theme-customization" className="scroll-mt-48" />

```tsx
const theme = {

      },
    },
  },
};
```
