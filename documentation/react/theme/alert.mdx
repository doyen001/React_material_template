# Tailwind CSS Alert - Theme

<span id="overview" className="scroll-mt-48" />

Learn how to customize the theme and styles for alert component, the theme object for alert component has three main objects:

**A.** The <Code>defaultProps</Code> object for setting up the default value for props of alert component.<br />
**B.** The <Code>valid</Code> object for customizing the valid values for alert component props.<br />
**C.** The <Code>styles</Code> object for customizing the theme and styles of alert component.<br />

You can customize the theme and styles of alert component by adding Tailwind CSS classes as key paired values for objects.

<br />
<br />

## Alert Theme Object Type

<span id="theme-object-type" className="scroll-mt-48" />

```ts
interface AlertStylesType {
  defaultProps: {
    variant: string;
    color: string;
    icon: node;
    show: boolean;
    dismissble: {
      action: node;
      onClose: func;
    };
    animate: {
      mount: object;
      unmount: object;
    };
    className: string;
  };
  valid: {
    variants: string[];
    colors: string[];
  };
  styles: {
    base: object;
    variants: {
      filled: object;
      gradient: object;
    };
  };
}
```

<br />
<br />

### For TypeScript Only

```tsx
import { AlertStylesType } from "@material-tailwind/react";
```

## Alert Theme Customization

<span id="theme-customization" className="scroll-mt-48" />

```ts
const theme = {
  alert: {
    styles: {
      variants: {
        filled: {
          blue: {
            backgroud: "bg-blue-500",
            color: "text-white",
          },
        },
        gradient: {
          blue: {
            backgroud: "bg-gradient-to-tr from-blue-600 to-blue-400",
            color: "text-white",
          },
        },
      },
    },
  },
};
```
