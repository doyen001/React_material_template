# Tailwind CSS Icon Button - Theme

<span id="overview" className="scroll-mt-48" />

Learn how to customize the theme and styles for icon button component, the theme object for icon button component has three main objects:

**A.** The <Code>defaultProps</Code> object for setting up the default value for props of icon button component.<br />
**B.** The <Code>valid</Code> object for customizing the valid values for icon button component props.<br />
**C.** The <Code>styles</Code> object for customizing the theme and styles of icon button component.<br />

You can customize the theme and styles of icon button component by adding Tailwind CSS classes as key paired values for objects.

<br />
<br />

## Icon Button Theme Object Type

<span id="theme-object-type" className="scroll-mt-48" />

```ts
interface IconButtonStyleTypes {
  defaultProps: {
    variant: string;
    size: string;
    color: string;
    ripple: boolean;
    className: string;
  };
  valid: {
    variants: string[];
    sizes: string[];
    colors: string[];
  };
  styles: {
    base: object;
    sizes: {
      sm: object;
      md: object;
      lg: object;
    };
    variants: {
      filled: object;
      gradient: object;
      outlined: object;
      text: object;
    };
  };
}
```

<br />
<br />

### For TypeScript Only

```tsx
import type { IconButtonStyleTypes } from "@material-tailwind/react";
```

## Icon Button Theme Customization

<span id="theme-customization" className="scroll-mt-48" />

```ts
const theme = {
  iconButton: {
    styles: {
      sizes: {
        sm: {
          fontSize: "text-xs",
        },
        md: {
          fontSize: "text-xs",
        },
        lg: {
          fontSize: "text-sm",
        },
      },
    },
  },
};
```
