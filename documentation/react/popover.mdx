# Tailwind CSS Popover - React

Our <Code>Popover</Code> component for Tailwind CSS & React is a small overlay of content that is used to demonstrate secondary information of any component when a user clicks it.

See below our simple <Code>Popover</Code> example that you can use in your web project.

<br />

<ComponentDemo id="popover">
  <Popover>
    <PopoverHandler>
      <Button variant="gradient">Show Popover</Button>
    </PopoverHandler>
    <PopoverContent>
      <span>This is a very beautiful popover, show some love.</span>
    </PopoverContent>
  </Popover>
</ComponentDemo>

```jsx
import {
  Popover,
  PopoverHandler,
  PopoverContent,
  Button,
} from "@material-tailwind/react";

export default function Example() {
  return (
    <Popover>
      <PopoverHandler>
        <Button variant="gradient">Show Popover</Button>
      </PopoverHandler>
      <PopoverContent>
        This is a very beautiful popover, show some love.
      </PopoverContent>
    </Popover>
  );
}
```

## Custom Popover Animation

You can modify the open/close state animation for <Code>Popover</Code> component using the <Code>animate</Code> prop.

<ComponentDemo id="custom-popover-animation">
  <Popover
    animate={{
      mount: { scale: 1, y: 0 },
      unmount: { scale: 0, y: 25 },
    }}
  >
    <PopoverHandler>
      <Button variant="gradient">Show Popover</Button>
    </PopoverHandler>
    <PopoverContent>
      <span>This is a very beautiful popover, show some love.</span>
    </PopoverContent>
  </Popover>
</ComponentDemo>

```jsx
import {
  Popover,
  PopoverHandler,
  PopoverContent,
  Button,
} from "@material-tailwind/react";

export default function Example() {
  return (
    <Popover
      animate={{
        mount: { scale: 1, y: 0 },
        unmount: { scale: 0, y: 25 },
      }}
    >
      <PopoverHandler>
        <Button variant="gradient">Show Popover</Button>
      </PopoverHandler>
      <PopoverContent>
        This is a very beautiful popover, show some love.
      </PopoverContent>
    </Popover>
  );
}
```
