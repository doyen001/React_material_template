# Tailwind CSS Icon Button - React

Use this example to create user-friendly buttons with icon for your Tailwind CSS and React project.

<br />

<ComponentDemo id="icon-button">
  <IconButton>
    <i className="fas fa-heart" />
  </IconButton>
</ComponentDemo>

```jsx
import { IconButton } from "@material-tailwind/react";

export default function Example() {
  return (
    <IconButton>
      <i className="fas fa-heart" />
    </IconButton>
  );
}
```

---

## Icon Button Variants

The <Code>IconButton</Code> component comes with 4 different variants that you can change it using the <Code>variant</Code> prop.

<ComponentDemo id="icon-button-variants">
  <div className="flex gap-4">
    <IconButton>
      <i className="fas fa-heart" />
    </IconButton>
    <IconButton variant="gradient">
      <i className="fas fa-heart" />
    </IconButton>
    <IconButton variant="outlined">
      <i className="fas fa-heart" />
    </IconButton>
    <IconButton variant="text">
      <i className="fas fa-heart" />
    </IconButton>
  </div>
</ComponentDemo>

```jsx
import { IconButton } from "@material-tailwind/react";

export default function Variants() {
  return (
    <div className="flex gap-4">
      <IconButton>
        <i className="fas fa-heart" />
      </IconButton>
      <IconButton variant="gradient">
        <i className="fas fa-heart" />
      </IconButton>
      <IconButton variant="outlined">
        <i className="fas fa-heart" />
      </IconButton>
      <IconButton variant="text">
        <i className="fas fa-heart" />
      </IconButton>
    </div>
  );
}
```

## Icon Button Sizes

The <Code>IconButton</Code> component comes with 3 different sizes that you can change it using the <Code>size</Code> prop.

<ComponentDemo id="icon-button-sizes">
  <div className="flex items-end gap-4">
    <IconButton size="sm">
      <i className="fas fa-heart" />
    </IconButton>
    <IconButton>
      <i className="fas fa-heart" />
    </IconButton>
    <IconButton size="lg">
      <i className="fas fa-heart fa-lg" />
    </IconButton>
  </div>
</ComponentDemo>

```jsx
import { IconButton } from "@material-tailwind/react";

export default function Sizes() {
  return (
    <div className="flex items-end gap-4">
      <IconButton size="sm">
        <i className="fas fa-heart" />
      </IconButton>
      <IconButton size="md">
        <i className="fas fa-heart" />
      </IconButton>
      <IconButton size="lg">
        <i className="fas fa-heart fa-lg" />
      </IconButton>
    </div>
  );
}
```

## Icon Button Ripple Effect

You can turn on/off the ripple effect for the <Code>IconButton</Code> component using the <Code>ripple</Code> prop.

<ComponentDemo id="icon-button-ripple-effect">
  <div className="flex w-max gap-4">
    <IconButton ripple={true}>
      <i className="fas fa-heart" />
    </IconButton>
    <IconButton ripple={false}>
      <i className="fas fa-heart" />
    </IconButton>
  </div>
</ComponentDemo>

```jsx
import { IconButton } from "@material-tailwind/react";

export default function Example() {
  return (
    <div className="flex w-max gap-4">
      <IconButton ripple={true}>
        <i className="fas fa-heart" />
      </IconButton>
      <IconButton ripple={false}>
        <i className="fas fa-heart" />
      </IconButton>
    </div>
  );
}
```
