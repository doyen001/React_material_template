# Tailwind CSS Button - Props

<span id="button-props" className="scroll-mt-48" />
The following props are available for button component. These are the custom props
that come with we've added for the button component and you can use all the other
native button props as well.

| Attribute   | Type                      | Description                            | Default                                |
| ----------- | ------------------------- | -------------------------------------- | -------------------------------------- |
| `variant`   | [Variant](#types-variant) | Change button variant                  | <Code>filled</Code>                    |
| `size`      | [Size](#types-size)       | Change button size                     | <Code>md</Code>                        |
| `color`     | [Color](#types-color)     | Change button color                    | <Code>blue</Code>                      |
| `fullWidth` | <Code>boolean</Code>      | Change button to a block level element | <Code>false</Code>                     |
| `ripple`    | <Code>boolean</Code>      | Add ripple effect for button           | <Code>true</Code>                      |
| `className` | <Code>string</Code>       | Add custom className for button        | `''`                                   |
| `children`  | <Code>node</Code>         | Add content for button                 | No default value it's a required prop. |

<br />
<br />

### For TypeScript Only

```tsx
import type { ButtonProps } from "@material-tailwind/react";
```
