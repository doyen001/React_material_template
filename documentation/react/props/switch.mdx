# Tailwind CSS Switch - Props

<span id="switch-props" className="scroll-mt-48" />
The following props are available for switch component. These are the custom props
that come with we've added for the switch component and you can use all the other
native input props as well.

| Attribute        | Type                  | Description                                   | Default                |
| ---------------- | --------------------- | --------------------------------------------- | ---------------------- |
| `color`          | [Color](#types-color) | Change switch color                           | <Code>blue</Code>      |
| `label`          | <Code>string</Code>   | Add label for switch                          | `''`                   |
| `ripple`         | <Code>boolean</Code>  | Add ripple effect for switch                  | <Code>true</Code>      |
| `className`      | <Code>string</Code>   | Add custom className for switch               | `''`                   |
| `containerProps` | <Code>object</Code>   | Add custom props for switch container         | <Code>undefined</Code> |
| `labelProps`     | <Code>object</Code>   | Add custom props for switch label             | <Code>undefined</Code> |
| `circleProps`    | <Code>object</Code>   | Add custom props for switch circle            | <Code>undefined</Code> |
| `inputRef`       | <Code>ref</Code>      | Add reference for <Code>input</Code> element. | <Code>undefined</Code> |

<br />
<br />

### For TypeScript Only

```tsx
import type { SwitchProps } from "@material-tailwind/react";
```
