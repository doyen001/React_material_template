# Tailwind CSS Avatar - Props

<span id="avatar-props" className="scroll-mt-48" />
The following props are available for avatar component. These are the custom props
that come with we've added for the avatar component and you can use all the other
native img props as well.

| Attribute   | Type                      | Description                     | Default              |
| ----------- | ------------------------- | ------------------------------- | -------------------- |
| `variant`   | [Variant](#types-variant) | Change avatar variant           | <Code>rounded</Code> |
| `size`      | [Size](#types-size)       | Change avatar size              | <Code>md</Code>      |
| `className` | <Code>string</Code>       | Add custom className for avatar | `''`                 |

<br />
<br />

### For TypeScript Only

```tsx
import type { AvatarProps } from "@material-tailwind/react";
```
